# Number Guessing Game

A simple Java CLI Number Guessing Game.

## Features

- Random number between 1 and 100
- Three difficulty levels
- Easy (10 chances)
- Medium (5 chances)
- Hard (3 chances)
- Timer
- Hint System
- High Score
- Play Multiple Rounds

## Requirements

- Java JDK 8 or above

## Compile

```bash
javac NumberGuessingGame.java
```

## Run

```bash
java NumberGuessingGame
```
